# Upflairs-Day4-Assignment
Anchor Tags & Lists
This repository contains all 5 completed tasks for the Day 4 Full Stack Web Development assignment.
It demonstrates internal page navigation using anchor IDs, table of contents bookmarking, and link protocols like mailto: and tel:.
The project showcases practical uses of Unordered (<ul>), Ordered (<ol>), Description (<dl>), and deeply nested list structures.
Additionally, it highlights differences between absolute and relative URLs for effective document linking.
All files are properly structured with clean HTML5 markup, individual CSS styling files, and well-organized codes.
