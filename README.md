# Upflairs-Day4-Assignment

Anchor Tags & Lists

This repository contains all 5 tasks for the Day 4 Full Stack Web Development assignment.
It covers smooth internal navigation using anchor tags, IDs, and link protocols like mailto: and tel:.
It showcases Unordered, Ordered, Description, and deeply nested lists for structured layouts.
It also highlights the practical differences between absolute and relative URLs.
All tasks are styled individually using clean HTML5 and custom CSS.
